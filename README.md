本仓库按模块整理了音控培训的基础内容、现场实操和团队协作规范。建议按以下顺序学习：

2. [Audio-Basics.md](Audio-Basics.md)
  - 声音、声学与数字音频基础

1. [Audio-Mixing.md](Audio-Mixing.md)

2. [Audio-Live-Sound-Reinforcement.md](Audio-Live-Sound-Reinforcement.md)

3. [Media-and-Projection.md](Media-and-Projection.md)

  - 诗歌、圣经与投影软件的基本使用

4. [Lighting.md](Lighting.md)

  - 灯光参数、色彩、氛围与常见操作

5. [Team-Software.md](Team-Software.md)
  - 团队软件、私有化服务与云端协作工具

6. [Standard-Operating-Procedures.md](Standard-Operating-Procedures.md)
  - 音控室推荐规范流程
  - 主日大堂音频与投影的日常操作顺序

## 说明

- 本仓库内容以教会音控服侍场景为主，既包含理论，也包含现场操作规范。
- 建议先看总纲，再按“音频基础 -> 混音 -> 现场扩音 -> 媒体投影 -> 灯光 -> 团队软件 -> SOP”的顺序学习。
